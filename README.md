PO POLSKU
----------
1. Otwórz link w przeglądarce (Chrome / Edge)
2. Kliknij przycisk  ▶ Skanuj
3. Poczekaj ~1-2 minuty (ładuje wszystkie rynki)
4. Gotowe!

CO OZNACZAJĄ KOLORY:
  Zielony / Oversold   = RSI < 30  mozliwe odbicie w gore
  Czerwony / Overbought = RSI > 70  mozliwy spadek
  Szary / Neutralny    = RSI 30-70  brak sygnalu

FILTRY:
  Sygnal    → tylko Oversold / Overbought / Blisko progu
  Kategoria → tylko Crypto / Commodities (Gold,Silver,Oil) / Equities (akcje)

UWAGA: RSI to tylko jeden wskaznik — nie jest to porada inwestycyjna!


IN ENGLISH
-----------
1. Open the link in your browser (Chrome / Edge)
2. Click the  ▶ Scan  button
3. Wait ~1-2 minutes (loading all markets)
4. Done!

WHAT THE COLORS MEAN:
  Green / Oversold    = RSI < 30  possible bounce up
  Red / Overbought    = RSI > 70  possible drop
  Grey / Neutral      = RSI 30-70 no signal

FILTERS:
  Signal   → show only Oversold / Overbought / Near threshold
  Category → show only Crypto / Commodities (Gold,Silver,Oil) / Equities (stocks)

NOTE: RSI is just one indicator — not financial advice!
